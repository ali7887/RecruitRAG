# RecruitRAG Vision

## Overview

RecruitRAG is an AI-powered hiring intelligence engine.

It analyzes resumes against job descriptions using:
- semantic search
- embeddings
- retrieval augmented generation (RAG)
- LLM reasoning

## Problem

Recruiters manually review hundreds of resumes.

This process is:
- slow
- repetitive
- inconsistent
- difficult to scale

Traditional ATS systems rely on keyword matching and miss contextual relevance.

## Solution

RecruitRAG:
1. extracts resume content
2. chunks text
3. generates embeddings
4. retrieves relevant sections
5. analyzes candidate-job fit using AI

## MVP Goals

- Upload PDF resumes
- Paste job descriptions
- Generate AI analysis
- Produce match scores
- Explain strengths and gaps
- Suggest interview questions

## Non Goals

The MVP will NOT include:
- authentication
- databases
- payments
- multi-user support
- dashboards
- analytics
